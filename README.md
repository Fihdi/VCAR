# 8hp Voltage Controlled Attack Release / Slew Rate Limiter

VCAR is based on the Serge Dual Universal Slope Generator (DUSG). In Envelope mode, VCAR acts as an Attack-Release Envelope. With LOOP engaged, the AR Envelope cycles indefinitely, allowing for VCO or LFO usage depending on the rate set by the Attack and Release settings. The looping can be stopped with an active Gate being present at the LOOP inhibit input. In Slew mode the output follows the input with a limited slew rate.

Use cases include Envelope Follower, Gate Delay, LFO/VCO, VCA, Wave Shaper, AR-Envelope, AHR-Envelope, Gate to Trigger Converter.
